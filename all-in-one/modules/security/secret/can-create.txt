

AWS Secrets Manager
    Managed service to securely store, retrieve, and rotate secrets like DB passwords, API keys, tokens.


What it solves
    Removes hard-coded secrets from code and Terraform state
    Centralized secret management
    Automatic rotation support

What you store
    Database credentials
    API keys / OAuth tokens
    App secrets (JWT secret, SMTP password)


How it works (simple)
    Create a secret (key–value or JSON)
    Grant IAM permission to app / service
    App fetches secret at runtime via AWS SDK
    Optional: enable automatic rotation



Where it’s used (industry standard)
    ECS / EKS
    App Runner
    Lambda
    EC2
    RDS credential rotation



Why it’s preferred
    Encrypted at rest (KMS)
    IAM-based access control
    Native AWS integration
    Auditable (CloudTrail)


One-line summary
    Secure secrets storage + runtime access + rotation. No secrets in code.
