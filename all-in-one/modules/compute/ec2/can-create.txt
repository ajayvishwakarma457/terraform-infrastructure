3. Storage (EC2-attached only)

    Attach / detach volumes
    Resize volumes
    Snapshot volumes

    Resources
        aws_ebs_volume
        aws_volume_attachment

    Private / public IP handling

    Resources
        aws_eip

5. Security (EC2 firewall & access)

    Security Groups (inbound/outbound)

    Resources
        aws_security_group
        aws_security_group_rule


6. Scaling (EC2 only)

    Launch Templates
    Auto Scaling Groups
    Instance refresh

    Resources
        aws_launch_template
        aws_autoscaling_group



7. Placement & Availability

    Availability Zone selection
    Placement groups (cluster / spread)
    Tenancy (default / dedicated)

    Resources
        aws_placement_group



8. Lifecycle Control

    Prevent destroy
    Force recreate on change
    Ignore changes (drift control)

    Terraform features
        lifecycle {}

9. Cost Control (EC2 scope)

    Resources
        instance_market_options
